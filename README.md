# Гражданская Оборона — Егор Летов

Статический сайт памяти Егора Летова и группы «Гражданская Оборона»: главная страница, раздел «Философия» и интерактивная дискография.

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub (например, `grob` или `grazhdanskaya-oborona`).
2. В каталоге этого проекта выполните:
   ```bash
   git remote add origin https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПО.git
   git branch -M main
   git push -u origin main
   ```
3. На GitHub откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Укажите ветку **main**, папку **/ (root)** и сохраните настройки.

Через несколько минут сайт будет доступен по адресу:

`https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/`

Имя репозитория входит в URL: если репозиторий называется `grob`, адрес будет `username.github.io/grob/`.

## Структура

- `index.html` — главная страница
- `philosophy/` — статья о философии Летова
- `discography/` — карусель альбомов и прослушивание
- `shared/`, `image/`, `fonts/`, `music/` — общие ресурсы
